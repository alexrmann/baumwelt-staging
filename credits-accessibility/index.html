Accessibility Statement

Measures to support accessibility

At the time of this site's publishing, Baumwelt LLC takes the following measures to ensure accessibility of this website, Baumwelt | Landscape Architects:

  • Include accessibility as part of our mission statement.
  • Include accessibility throughout our internal policies.

Conformance status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. At this time, this website is not assessed with WCAG. Not assessed means that the content has not been evaluated or the evaluation results are not available.

Browser and Device Compatibility

It is our aim to provide a web experience compatible with screen-readers and other assistive technologies. We anticipate all features of this site will function as expected for all browsers included in the W3C WebDX Community Group "Baseline" designation.

Users of the following browsers and hardware may experience issues:

  • Internet Explorer 9 and earlier
  • Older iOS and Android smartphones

Technical specifications

Accessibility of this website relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • WAI-ARIA
  • CSS
  • JavaScript

These technologies are relied upon for conformance with the accessibility standards used.

Limitations and alternatives

Despite our best efforts to ensure accessibility of this website, there may be some limitations. While our goal is to provide a universally accessible user experience, this site is a work in progress, and thus we cannot yet guarantee that our goals have been reached. We continue to learn and integrate web accessibility best practices whenever and wherever possible. If you encounter problems viewing content or otherwise interacting with the website, or if you have feedback about this statement, please let us know by sending us an e-mail.

Comments & Feedback

We welcome your feedback on the accessibility of Baumwelt | Landscape Architects. Please let us know if you encounter accessibility barriers on this website.
