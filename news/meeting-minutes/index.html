Review Markdown Guide

Site content, including news and projects, should conform to Markdown syntax whenever possible. Jekyll (our site generator) interprets Markdown files (.md) and converts them to HTML.

Writing files with Markdown syntax is the simplest way to add new content to the site. Markdown is easy to learn and use. Visit Markdown Guide for instructions.

  • Start here: Getting Started
  • Most content can be created with the Basic Syntax.
  • More complex formatting options are outlined in the Extended Syntax

If this is too cumbersome, I am able to convert Word documents (.docx) to Markdown using a tool called Pandoc. Any images should be included in the Word file. We’ll need to test this out to be sure it will be a functional alternative to writing .md files directly.

  • For more information see How to Convert Word Documents to Markdown: Complete Guide
